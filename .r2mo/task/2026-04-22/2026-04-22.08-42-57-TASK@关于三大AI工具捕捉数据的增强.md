三个基础的AI工具：
- Codex CLI
- Claude Code CLI
- OpenCode CLI
目前在本地仓库和远程仓库中可能都有正在运行的 session 或打开的 session，尝试通过捕捉强化实时监控板，核心在于：
1. Terminal 要可以捕捉
2. 会话ID要可以捕捉
3. 状态要可以捕捉
4. 后续的就绪时间和工作时间也要可以捕捉
目前上述四个功能出现了不稳定或不完整的情况，在线搜索一下相关解决方案，统一处理这个问题。
